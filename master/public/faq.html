If anything is unclear: [sequencing@fgcz.ethz.ch](mailto:sequencing@fgcz.ethz.ch?subject=SUSHI) # How to subset samples? There are two possibilities: - After starting an analysis you can use "samples" selection box to select/unselect the samples. This will not modify the parent DataSet but only the Child DataSet to be generated.![](Images/samples-parameter.png) - On a dataset, clicking on "edit table" brings you to the "edit screen" that lets you remove samples from the DataSet using the minus signs on the left. Subsequently do "save as child" to generat a Child DataSet with the subset of samples. Clickin on "save" will permanently modify the current DataSet.![](Images/subset-dataset.png)
